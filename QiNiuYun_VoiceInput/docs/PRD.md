# 语音输入法 — 产品需求文档 (PRD)

## 1. 产品定位
一个基于 Web 的智能语音输入工具，用户通过浏览器录音，后端调用语音识别引擎（七牛云 API / Vosk 离线）将语音转为文字，并提供文本编辑、历史记录等能力。目标是**帮助用户提高文本输入效率**。

## 2. 用户场景
- 写文章/报告时，口述比打字更快
- 双手被占用（做饭等）时，用语音记录灵感
- 长文本输入场景，减少打字疲劳

## 3. 功能需求

### 3.1 核心功能

| 模块 | 功能 | 描述 | 优先级 |
|------|------|------|--------|
| 语音录制 | 按住录音 | 按住按钮录制，松开自动停止并上传 | P0 |
| 语音录制 | 点击录音 | 点击开始/停止录音 | P0 |
| 语音录制 | 录音状态指示 | 录音中动画、音频波形 | P1 |
| 语音识别 | 七牛云识别 | 上传音频到七牛云 API，返回文字 | P0 |
| 语音识别 | Vosk 离线识别 | 本地 Vosk 模型识别（离线可用） | P1 |
| 语音识别 | 引擎切换 | 用户可手动切换识别引擎 | P1 |
| 文本编辑 | 文本展示 | 展示识别结果 | P0 |
| 文本编辑 | 光标编辑 | 支持在识别结果中移动光标、插入、删除 | P0 |
| 文本编辑 | 复制全部 | 一键复制所有文本到剪贴板 | P1 |
| 文本编辑 | 清空文本 | 一键清空当前文本 | P1 |
| 文本编辑 | 标点优化 | 智能添加标点符号（七牛云默认支持） | P1 |
| 历史记录 | 查看历史 | 按时间倒序查看历史识别记录 | P1 |
| 历史记录 | 复用历史 | 点击历史记录重新加载到编辑器 | P2 |

### 3.2 非功能需求

| 维度 | 要求 |
|------|------|
| 响应速度 | 录音结束后 3s 内返回识别结果（七牛云 API） |
| 准确度 | 普通话识别准确率 > 90% |
| 并发 | 支持 50 人同时使用 |
| 易用性 | 打开浏览器即用，无需安装 |
| 成本 | 七牛云 API 按量计费，Vosk 离线免费 |

## 4. 业务流程

```
用户打开页面 → 点击录音按钮 → 浏览器获取麦克风权限
    → 开始录音（显示录音动画）
    → 停止录音 → 上传音频到后端
    → 后端调用 ASR 引擎（七牛云/Vosk）
    → 返回识别文本 → 前端展示在编辑器
    → 用户可编辑 / 复制 / 保存
```
